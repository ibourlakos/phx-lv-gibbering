# #1 · Establish git remote

**Status:** open
**Opened:** 2026-06-03
**Priority:** high
**Tags:** ops

Connect the local repository to a remote so the team can push and collaborate.

**Acceptance criteria**
- [ ] Remote repository created (GitHub / GitLab / self-hosted Gitea / etc.)
- [ ] `git remote add origin <url>` configured locally
- [ ] `git push -u origin main` succeeds
- [ ] Remote URL added to `docs/dev-setup.md` under First-Time Setup
- [ ] Git LFS remote storage verified (if using GitHub: included free; others may need separate config)
