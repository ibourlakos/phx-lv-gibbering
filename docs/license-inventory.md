## LPC (Liberated Pixel Cup) assets

Any LPC sprite committed here must use the **CC-BY-SA 3.0** license option only. See the LPC ruling in [docs/legal.md § LPC ruling](legal.md#lpc-liberated-pixel-cup-ruling--cc-by-sa-only-gpl-20-option-prohibited).

When adding an LPC asset, add a row to the table above with:
- **License:** CC-BY-SA 3.0
- **Source:** full OpenGameArt.org URL for the specific asset
- **Artist / Attribution:** name(s) listed on the OGA page
- **Obligations:** attribute in-game or in README; no DRM on sprite files; modified sprites stay CC-BY-SA
